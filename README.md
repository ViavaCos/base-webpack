# base-webpack
webpack5的基本配置 (A base configuration with webpack 5.)

(PS: 包含大量注释，若欲学习，可通过查看提交记录更为方便)

- [x] https://github.com/ViavaCos/base-webpack/commits/main



## 使用(Usage)

```bash
# 1.安装依赖
npm i

# 2.启动开发服务器
webpack serve
```
